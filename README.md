# Lucid-Web-Components
A playground for web components that solve real world common ui problems

## Getting Started

### Download Repository
`git clone https://github.com/alucidwolf/Lucid-Web-Components.git`

### Install Dependencies
`npm install`

### Run Project
`npm start`
